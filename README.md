## 🚥 프로젝트 특징
✔️ **깔끔한 UI/UX**: Figma로 설계된 직관적이고 아름다운 디자인  
✔️ **직관적인 자격증 캘린더/상세 페이지**: 번거로운 검색 없이 직관적으로 확인  
✔️ **꿀팁 공유 족보게시판**: 실제 응시자가 작성한 시험 관련 팁, 정보                               
✔️ **실전같은 문제게시판**: 성적을 확인하고 즉시 피드백이 가능
